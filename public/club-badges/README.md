Сюда нужно загружать эмблемы клубов для выбора в турнире.

Как это работает:
- имя файла становится slug клуба
- из slug автоматически собирается название клуба
- файл должен быть в формате `.png`, `.jpg`, `.jpeg`, `.webp` или `.svg`

Примеры:
- `real-madrid.png`
- `barcelona.png`
- `manchester-city.svg`

После загрузки новых эмблем клубы появятся в выборе при регистрации и в рандомном распределении.
